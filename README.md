Ragnarok Online Char Generator 

*inspired by https://ro-character-simulator.ratemyserver.net/*

```
pnpm i
pnpm dev
```

Query params: 
- Head (01,02,03)
- Body (LK,MG)
- Wing (gabriel,bloody)
- Hat (wildcat_knit_cap,white_corone,cowboy_fire)


```
http://localhost:3000/generate?head=02&body=MG&wing=gabriel&hat=flapping_angel_wing
```
